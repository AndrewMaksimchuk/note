# Get bytecode of file/function  

Examples: 

All file  
``` sh
node --print-bytecode --print-code-verbose ./index.mjs > bytecode
```

Show only function 'testFunction' in 'index.mjs' file  
``` sh
node --print-bytecode --print-bytecode-filter=testFunction ./index.mjs
```
