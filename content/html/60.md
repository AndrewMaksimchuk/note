# Tag <ins>

Defines a text that has been inserted into a  
document

## Attributes

- `cite`
- `datetime`

## Example

```html
<p>“You're late!”</p>
<del>
  <p>“I apologize for the delay.”</p>
</del>
<ins cite="../howtobeawizard.html"
datetime="2018-05">
  <p>“A wizard is never late …”</p>
</ins>
```
