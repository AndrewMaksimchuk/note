# Tag <table>

Defines a table

`display: table;`  

```html
<table>
  <thead>
    <tr>
      <th>Town</th>
      <th>State</th>
      <th>Population</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Candy Kitchen</td>
      <td>New Mexico</td>
      <td>59</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>First Column Footer</td>
      <td>Second Column Footer</td>
      <td>Yet Another Column Footer</td>
    </tr>
  </tfoot>
</table>
```
