# Tag <area>

Defines an area inside an image map that has predefined clickable areas. 
An image map allows geometric areas on an image to be associated with hypertext links.
This element is used only within a <map> element.

`display: inline;`  

## Examples

```html
<map name="primary">
  <area
    shape="circle"
    coords="75,75,75"
    href="left.html"
    alt="Click to go Left" />
  <area
    shape="circle"
    coords="275,75,75"
    href="right.html"
    alt="Click to go Right" />
</map>
<img
  usemap="#primary"
  src="https://dummyimage.com/350x150"
  alt="350 x 150 pic" />
```
