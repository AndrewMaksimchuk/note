[attribute]
    [target]
    Selects all elements with a target attribute

``` css
a[title]
a[href="https://example.org"]
a[href*="example"]
a[href$=".org" i]
a[class~="logo"]
a[href^="#"]
a[href*="cAsE" s]
a[href*="insensitive" i]
a[href^="https://"][href$=".org"]
div:not([lang])
div[lang="pt"]
```
