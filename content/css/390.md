# The Good Old margin: auto with Flexbox

Combined with flexbox, margin: auto can center a
flex item horizontally and vertically very easily.

```html
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 300px;
  height: 200px;
  background: #ccc;
  display: flex;
}

.child {
  width: 50px;
  height: 50px;
  background: #000;
  margin: auto;
}
```
