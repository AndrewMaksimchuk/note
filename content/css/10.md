# Key word max-content

`max-content` - ключове слово що представляє  
власну максимальну або ширину або висоту  
контента(вмісту).  

```css
/* Used as a length */
width: max-content;
inline-size: max-content;
height: max-content;
block-size: max-content;

/* used in grid tracks */
grid-template-columns: 200px 1fr max-content;
```
