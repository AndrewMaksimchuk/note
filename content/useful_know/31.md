# MCC-код (Merchant Category Code)

MCC-код (Merchant Category Code) — це чотиризначний номер, який присвоюється торговій точці або сервісу для визначення типу її діяльності під час обробки платежів банками чи платіжними системами (наприклад, Visa або Mastercard).

Простіше кажучи, MCC показує, за що саме ти платиш.

Приклади MCC-кодів:
5411 — супермаркети
5812 — ресторани та кафе
4111 — транспорт (таксі, громадський транспорт)
5912 — аптеки
5999 — різні роздрібні магазини

Для чого він потрібен:
банки використовують MCC для нарахування кешбеку (наприклад, більше % за ресторани);
визначення, чи підпадає операція під комісію або пільги;
аналіз витрат у мобільних банках;
іноді — для блокування або обмеження платежів.

Де подивитися MCC:
у деталях транзакції в банківському додатку;
через службу підтримки банку;
іноді на чеку або в інтернет-магазині.
