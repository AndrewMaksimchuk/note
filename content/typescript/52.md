# tsconfig.json option - compilerOptions

In a **tsconfig.json**, the types field of **compilerOptions** specifies a list of package names to be included in the global scope during compilation. 
Typically, packages in node_modules are automatically included via imports in your source code; 
but for convenience, TypeScript would also include all packages in **node_modules/@types** by default, 
so that you can get global declarations like process or the "fs" module from **@types/node**, 
or describe and it from **@types/jest**, without needing to import them directly.

```typescript
  {
      "compilerOptions": {
          // Explicitly list the @types packages you need
         "types": ["node", "jest"]
      }
  }
```
