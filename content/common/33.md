# doiuse - Lint CSS for browser support against Can I use database

## Example

```sh
doiuse --browsers "ie >= 9, > 1%, last 2 versions" main.css
cat main.css | doiuse --browsers "ie >= 9, > 1%, last 2 versions"
```
